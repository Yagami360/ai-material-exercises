# NVIDIA Clara for Biopharma / BioNeMo の概要

https://www.nvidia.com/ja-jp/clara/biopharma/

NVIDIA Clara for Biopharma は、NVIDIA Clara における AI 創薬のための機能。
BioNeMo と呼ばれるフレームワークと AI 創薬のための事前学習済みモデル郡で構成される

```
┌─────────────────────────────────────────────┐
│  1. BioNeMo Framework (フレームワーク層)      │
│     - モデル訓練・推論エンジン                  │
│     - データローダー・前処理パイプライン          │
│     - 分散学習インフラ                        │
├─────────────────────────────────────────────┤
│  2. Pre-trained Models (モデル層)            │
│     - ESM-1, ESM-2 (タンパク質)              │
│     - MegaMolBART (分子生成)                 │
│     - ProtT5, ProtBERT (配列解析)            │
├─────────────────────────────────────────────┤
│  3. BioNeMo Service (クラウドサービス層)       │
│     - API エンドポイント                      │
│     - モデルホスティング                       │
│     - ファインチューニングサービス              │
└─────────────────────────────────────────────┘
```

## AI 創薬のための事前学習モデル郡

- AlphaFold2 (OpenFold)

    - Transformer ベースのモデルで、タンパク質の3次元構造を非常に高い精度で予測するAIモデル
        - タンパク質配列を「文章」として学習し、アミノ酸の並びパターンを理解
    - **入力**:<br>
        - アミノ酸配列
            - 予測したいタンパク質を構成するアミノ酸の並び（例：「M-G-V-A-L...」）です。
        - xxx
    - **出力**:<br>
        - タンパク質の3D構造<br>
            **PDB（Protein Data Bank）形式**で出力<br>
            - タンパク質を構成する各原子（主鎖と側鎖）の3次元座標（x, y, z）のリスト
        - **信頼度スコア**<br>


- ProteinMPNN
    - グラフ畳み込み（GNN）ベースのモデルで、タンパク質の３D構造からアミノ酸配列に逆変換するモデル
        - 科学者やエンジニアが「こういう機能を持つタンパク質が欲しい」と考えたとき、まず「こういう**形**（構造）が必要だ」と設計図を描くことがあります。ProteinMPNNは、その**設計図（構造）を実現するための「部品リスト」（アミノ酸配列）** を自動で生成してくれるツール
    - **入力**:
        - タンパク質の3D構造
    - **出力**:
        - 入力されたタンパク質の3D構造に折りたたまる可能性が高いアミノ酸配列

- RFdiffusion
    - 拡散モデルベースのモデルで、ノイズ入力から徐々にタンパク質構造を生成
        - **無制約生成 (Unconditional Generation):** 特定の機能や制約を設けず、完全に新しい、自然なタンパク質の骨格構造をランダムに生成する。
        - **条件付き生成 (Conditional Generation):** 特定の要件（例：特定の標的分子にくっつく形、環状の形、既存のタンパク質に挿入されるポケットの形など）を満たすように、構造を設計・生成する。
    - **入力**:
        - ノイズ入力
    - **出力**:
        - xxx

- ESM-1 / ESM-2 (進化的スケールモデリング)
    - Transformer ベースのモデルで、タンパク質配列を「文章」として学習し、アミノ酸の並びパターンを理解
    - **入力**: アミノ酸配列（例: MKWVTFISLLFSSAYS...）
    - **出力**: 3D構造予測、機能予測、変異影響予測

- モデルの組み合わせ

    ```
    1. RFdiffusion (創造): 「こういう機能に最適な構造をゼロから生成する。」
    2. ProteinMPNN (翻訳): 「この生成された構造を安定させるためのアミノ酸配列を設計する。」
    3. AlphaFold2 (検証): 「ProteinMPNNが設計した配列が、本当に意図した構造に折りたたまれるかを確認する。」
    ```


## 創薬の基本ドメイン知識


- なぜタンパク質が創薬の中心なのか
    - そもそも人体とは「タンパク質の集合体」

        ```
        人体の構成：
        ├─ 水分（60%）
        ├─ タンパク質（20%） ← 機能の主役
        ├─ 脂質（15%）
        ├─ ミネラル（5%）
        └─ 糖質他（1%）

        タンパク質の数：
        - 人体には約10万種類のタンパク質
        - 各細胞に約80億個のタンパク質分子
        - 体重60kgの人で約10kg分がタンパク質
        ```

    - 薬、病気、タンパク質の関係

        薬は、タンパク質の形にぴったりはまるような薬の分子がくっついて、タンパク質の働き方を変えることで病気を治す。もしくは、タンパク質が医薬品そのものである

- AI創薬の必要性

    - 課題
        - タンパク質は、20種類のアミノ酸の組み合わせで構成されるが、20種類のアミノ酸の組み合わせ数は天文学的数
        - また、タンパク質の3D構造が薬の作用にとって重要であるが、３D構造と薬（タンパク質）の機能の関係も複雑

- Protein Binder（タンパク質結合分子）

    - 特定の分子と結合する能力を持つタンパク質（薬となりえる）や、複数のタンパク質を結合させる架橋試薬などのこと
